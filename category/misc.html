xubuntu12.04 折腾记

二 28 八月 2012
By Mocker

In misc.

tags: xubuntu12.04 y450

折腾来由

本人持有y450，俗称小y。y恰好是姓氏到第一个字母。当然，这跟我当初为什么买小y没有一毛钱到关系。
电脑上一直装到ubuntu10.04，暑假回家到时候，心血来潮地折腾了下xfce4，其简单性，轻量性是我放弃gnome2催化剂。一不做，二不休，这次直接换装xubuntu了。选择了最新的12.04版本，不要误会，本人完全没有最新情结。

安装

硬盘安装比较麻烦，手头又没有光盘，借了个U盘，挂载了xubuntu。usb启动，安装........
由于以前，我把home目录单独挂载了一个分区，因此此次重装系统，不会影响到我到主目录，连备份都省了

开始折腾

安装好了后，界面很清新，有一些细节需要自己动手配置下
1.桌面图标字体背景不透明
在主目录下新建.gtk-2.0.rc文件
vim ~/.gtk-2.0.rc
写入一下内容
style "xfdesktop-icon-view" {
font_name = "sans Bold" #字体名称设置
XfdesktopIconView::label-alpha = 10 #透明度设置
base[NORMAL] = "#000000" #背景颜色(下同)
base[SELECTED] = "#71B9FF"
base[ACTIVE] = "#71FFAD"
fg[NORMAL] = "#ffffff" #文字颜色(下同)
fg[SELECTED] = "#71B9FF"
fg[ACTIVE] = "#71FFAD" }
widget_class "XfdesktopIconView" style "xfdesktop-icon-view"

2.更新源 新德立软件中心，选最佳服务器即可
由于是12.04,版本比较新，网上提到源到博客比较少。没必要手动写入
然后sudo apt-get update && sudo apt-get upgrade

3.vim sudo apt-get install vim后，发现不能共享系统剪切板。google后才知道是xterm-clipboard没有被编译进来。
难道要我自己编译？算了吧，直接sudo apt-get install vim-gtk.毕竟不是pcman，问题又来了,包到依赖怎么也弄不好
我都懒得说这心酸到过程了。。。。
直接说解决方法吧:
sudo apt-get autoremove vim-common sudo apt-get install vim-gtk 搞定！！！

4.显卡 开源的nvidia显卡可以让你电脑温度暴高，转速暴快
去官网下载一个最新的闭源显卡驱动
关掉x server:
xfce4 没有gdm
sudo service lightdm stop
然后Alt+F1 找到驱动所在目录,sudo sh ./xxxxx.run
最后生成下配置文件:sudo nvidia-xconfig
重启xserver：sudo service lightdm start

5.thunar启动慢
sudo vim /usr/share/gvfs/mounts/network.mount 更改AutoMount=false

6.无线网 重启，F2进BIOS,F9恢复默认设置，F10保存并重启
sudo ifconfig wlan0 up

7.小插件 sudo apt-get install xfce4-goodies 全部装了吧，不差那点内存

看到如此凌乱到博客，你大概知道我有多折腾了。。。。
